Екі өлшемді массив

Оқу мақсаты: екі өлшемді массивтер пайдаланып Python программалау тілінде программаларды жасау

Матрица

Матрица - (неміс тілінен аударғанда "matrise", латын тілінде "matrix" - "аналық") - математикада кез-келген жиынның элементтерінен құрылған және m жол мен n бағаннан тұратын тіктөртбұрышты кесте. Матрицаны түзетін нысандар оның элементтері деп аталады. Математикада және аналитикалық геометрияда матрицаларды (екіөлшемді кесте, массив) пайдаланып көптеген мәселелерді шешуге болады. Программалауда түрлі салаларда туындаған көптеген мәселелердің шешімі матрицалардың көмегімен табылады.

Екі өлшемді тізім құру

Тізімдер тізімі, мысалы, сандар тізіміне ұқсас екенін түсіну керек. Бағдарламада тізімдер тізімін көрсету үшін біз элементтерді төртбұрышты жақшаға үтірмен тізімдейміз: table = [[1, 2, 3], [4, 5, 6], [7, 8, 9]]

Көрнекілік үшін оны келесі түрде жазуға болады:

table = [[1, 2, 3],

[4, 5, 6],

[7, 8, 9]]

Бұл мысалда элементтер матрицасы (кесте түрінде) анық көрінеді. Берілген "кестедегі" элементпен байланысу үшін алдымен бірінші деңгей элементіне жүгіну керек.

Мысалы, table[1] = [4, 5, 6].

Екінші индекс кіріктірілген тізім элементтеріне жүгінуге мүмкіндік береді.

Мысалы, table[1][2] =

Матрица үшін бірінші деңгей индекстері қатар нөмірі, ал екінші деңгей индекстері баған нөмірі деп аталады. Қатар мен баған нөлден бастар индекстеледі.

Екі өлшемді тізімнің элементтерімен жұмыс. Тізімді экранға шығару

Тізімнің элементінмен байланысу үшін сол элементтің индексін тік жақшада көрсету керек. Екі өлшемді кіріктірілген тізімді қарастырғанда екі индексті көрсету қажет (әрқайсысы жеке квадрат жақшада). Екі өлшемді массивте жол нөмірі, содан кейін баған нөмірі көрсетіледі (алдымен кіріктірілген тізім, содан кейін оның элемент таңдалады).

Мысалы, экранға не шығатынын анықтаңыз:

table = [[1, 2, 3], [4, 5, 6], [7, 8, 9]]

print(table[0][0], table[0][1], table[1][0])
